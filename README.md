## Running

Clone and install dependencies:
```bash
$ npm install
```

Then run
```bash
$ npm start
```

And run webpack to hook up with your changes
```bash
$ webpack -w
```
or
```bash
$ node node_modules/webpack/bin/webpack.js -w
```
in case you have no global installation of webpack.
